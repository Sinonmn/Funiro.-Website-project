### Über das Projekt 

Ein vollständig responsiver Online-Shop für Möbel, komplett ohne Frameworks – nur **HTML5, CSS3 und modernes Vanilla JavaScript (ES6+)** entwickelt.

**Wichtige Features**
- Voll funktionsfähiges Burger-Menü mit Overlay und Scroll-Sperre  
- Nested Dropdown-Menü (mobile + desktop)  
- Hero- und Tipps-Slider mit **Splide.js** (autoplay, responsive Breakpoints)  
- Produktkarten mit Hover-Effekt (Desktop) und Tap-to-Open (Mobile)  
- Warenkorb-Funktionalität mit **localStorage**-Zähler im Header  
- 100 % responsive (Mobile → Tablet → Desktop)  
- Barrierefreiheit und Touch-Optimierung bewusst berücksichtigt

**Tech-Stack**
- HTML5 
- CSS3 
- Vanilla JavaScript
- Splide.js 
- localStorage
